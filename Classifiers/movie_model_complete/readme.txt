To train sentiment classifier for movies :

	1)download the train data from http://ai.stanford.edu/~amaas/data/sentiment/
	2)extract the downloaded file and keep the folder aclImdb in the same folder as the below files
	3)run prepare_data_movie.ipynb
	4)run pre_processing_movie.ipynb
	5)run ml_model_movie.ipynb

The above steps will create a classifer movies.pickle 


To predict: 
	run the classifiers/predict.ipynb file by giving the review and appropriate review type
